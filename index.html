Summer Projects 2025

AUTONOMOUS NAVIGATION

Developing a fully autonomous mobile robot using the TortoiseBot platform. Objective is to explore an unknown environment, create a real-time map using SLAM, and navigate efficiently to user-defined goals using path planning algorithms in ROS.

Ball Balancing Bot

Building a robotic system to balance a ball on a flat platform using two servo motors and Raspberry Pi. A camera tracks the ball’s position using computer vision and a PID algorithm keeps the ball centered.

HEXAPOD

Designing and testing the locomotion of a hexapod robot using RC communication. Implements gait patterns for smooth walking and climbing with stability.

Mechanical Turk

A robot designed to play chess autonomously — inspired by the historical 'Turk', but built with real engineering, no hidden chess master underneath.

StepXplorer

A stair-climbing robot with ultrasonic sensors to detect and climb stairs. Built with a stable platform to carry objects even over rough terrain.
